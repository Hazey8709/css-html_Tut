HTML Quick - Keys

(Warning there are spaces between < > to display HTML code)

ID & Class Attributes

1: #header

< div id="header">< /div>

2: .title

< div class="title">< /div>

3: p.class1.class2.class3

< p class="class1 class2 class3">< /p>


Multiplication

4: ul>li*5
- OR -
ul>li{listItem}*5 

            < ul>
                < li>< /li>
                < li>< /li>
                < li>< /li>
                < li>< /li>
                < li>< /li>
            < /ul>

Name & Numbering: $

5: ul>li.sample$*5

            < ul>
                < li class="sample1">< /li>
                < li class="sample2">< /li>
                < li class="sample3">< /li>
                < li class="sample4">< /li>
                < li class="sample5">< /li>
            < /ul>

6: h$[title=topic$]{Headline $}*3

            < h1 title="topic1">Headline 1< /h1>
            < h2 title="topic2">Headline 2< /h2>
            < h3 title="topic3">Headline 3< /h3>

                < li class="item003">< /li>
                < li class="item004">< /li>

            < ul>

Custom Attributes

10: p[title="Hello World"]

            < p title="Hello World">< /p>

Text{}:

11: a{Click Me}

            < a href="">Click Me< /a>

12: p>{Click }+a{here}+{ to continue}

            < p>Click < a href="">here< /a> to continue< /p>

HTML:

13: a

            < a href="">< /a>

14: a:link

            < a href="http://">< /a>

15: a:mail

            < a href="mailto:">< /a>

16: link

            < link rel="stylesheet" href="">

17: link:css

            < link rel="stylesheet" href="style.css">

18: meta:utf (Meta-UTF Code)

            < meta http-equiv="Content-Type" content="text/html;charset=UTF-8">

19: meta:vp (Meta-Viewport)

            < meta name="viewport" content="width=device-width, initial-scale=1.0">

20: meta:compat

            < meta http-equiv="X-UA-Compatible" content="IE=7">
